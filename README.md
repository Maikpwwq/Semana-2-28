# Semana-2-28

## Proyecto final Sprint Semana II

Sitio con componentes reutilizables de VUE y uso APIs para consulta de datos. El desarrollo del sitio web consta de 5 secciones

- Sección superior, cabecera o header.
- Sección de servicios.
- Sección de noticias.
- Sección de equipo.
- Sección Footer.
